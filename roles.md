# Team Roles

- **Data Engineer:** Owns data ingestion, validation, and feature pipeline reliability.
- **ML Engineer:** Designs modeling strategy (LightGBM ensemble, hashing, feature crosses), manages experiments, and model registry flows.
- **DevOps Engineer:** Maintains CI/CD (GitLab), containerization, deployment scripts, and infrastructure reliability.
- **Monitoring Lead:** Oversees CME, drift detection, alerting, and fallback/rollback governance using MLflow registry transitions.
